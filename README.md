# 🎵 SoundDrop

Descarga audio de YouTube gratis. Hecho con Node.js + Express + yt-dlp.

---

## 🚀 Cómo subir gratis a Railway

### Paso 1 — Crea una cuenta gratis
- Ve a https://github.com y crea una cuenta (si no tienes)
- Ve a https://railway.app y entra con tu cuenta de GitHub

### Paso 2 — Sube el código a GitHub
1. En GitHub, crea un repositorio nuevo llamado `sounddrop`
2. Sube todos estos archivos al repositorio

### Paso 3 — Despliega en Railway
1. En Railway, haz clic en **"New Project"**
2. Selecciona **"Deploy from GitHub repo"**
3. Elige tu repositorio `sounddrop`
4. Railway lo detecta automáticamente y lo despliega

### Paso 4 — Obtén tu URL
- Railway te da una URL tipo: `https://sounddrop-production.up.railway.app`
- ¡Esa es tu app! Funciona desde cualquier celular o computadora.

---

## 💻 Correr localmente

```bash
npm install
npm start
```

Abre http://localhost:3000
